*** Présentation

Ce projet a été réalisé dans le cadre du TME *Arithmétique et Logarithme Discret* de l'UE *Introduction à la Cryptologie (LU3IN024)* à Sorbonne Université.

L'objectif est d'implémenter en Python les principaux algorithmes d'arithmétique modulaire et de théorie des nombres utilisés en cryptographie moderne.

Le projet aboutit à une implémentation complète de l'algorithme *Baby-Step Giant-Step (BSGS)* permettant de résoudre le problème du logarithme discret (*Discrete Logarithm Problem - DLP*) dans un groupe cyclique fini.

**Le développement suit une approche progressive :

* implémentation des outils fondamentaux d'arithmétique modulaire ;
* construction de groupes cryptographiques ;
* génération de paramètres adaptés ;
* résolution du logarithme discret avec un algorithme efficace.

- Objectifs du projet

**Le projet vise à implémenter :

* l'algorithme d'Euclide étendu ;
* les coefficients de Bézout ;
* les inverses modulaires ;
* l'indicatrice d'Euler φ(n) ;
* l'exponentiation modulaire rapide ;
* la factorisation d'un entier ;
* le calcul de l'ordre d'un élément ;
* la recherche d'un générateur d'un groupe cyclique ;
* la génération de nombres premiers sûrs ;
* l'algorithme Baby-Step Giant-Step pour résoudre le logarithme discret.

À la fin du projet, le programme permet de construire un groupe adapté à la cryptographie puis de résoudre un problème de logarithme discret.

- Organisation du travail

Le TME est construit de manière incrémentale.

*** Exercice 1 — Arithmétique de base

But:Implémentation des fonctions fondamentales utilisées pour les calculs cryptographiques.

*** Fonctions

* calcul du PGCD ;
* calcul des coefficients de Bézout ;
* calcul des inverses modulaires ;
* calcul de l'indicatrice d'Euler.

Ces fonctions constituent la base de tous les calculs réalisés dans le projet.

** Exercice 2 — Logarithme discret

But: Manipulation du groupe multiplicatif modulo un nombre premier et préparation de l'algorithme de résolution du logarithme discret.

*** Fonctions

* exponentiation modulaire rapide ;
* factorisation de p-1 ;
* calcul de l'ordre d'un élément ;
* recherche d'un générateur du groupe.

** Exercice 3 — Génération de nombres premiers sûrs

But: Génération de paramètres cryptographiques adaptés aux groupes utilisés pour le logarithme discret.

Un nombre premier sûr est de la forme :

#+BEGIN_SRC text
p = 2q + 1
#+END_SRC

où q est également premier.

*** Fonctions

* génération d'un nombre premier ;
* test de primalité ;
* génération d'un nombre premier sûr d'une taille donnée.

** Exercice 4 — Baby-Step Giant-Step

But: Implémentation de l'algorithme de Shanks permettant de résoudre efficacement le problème du logarithme discret.

Principe :

1. Calcul des Baby Steps.
2. Calcul des Giant Steps.
3. Recherche d'une collision.
4. Reconstruction de l'exposant recherché.

La complexité passe d'une recherche exhaustive en O(n) à O(√n).
